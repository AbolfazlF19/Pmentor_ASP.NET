# Pmentor (ASP.NET)

Pmentor is a course sales web application developed using ASP.NET (non-Core).
The project focuses on providing a platform for managing and selling educational courses.

> ⚠️ Note: This project is still under development. Some features such as the payment gateway integration, email-based code delivery, and a few minor functionalities are not yet completed.

## 🔧 Technologies Used

* ASP.NET (Framework)
* C#
* SQL Server

## 📌 Features

* Course listing and management
* User interaction with courses
* Basic structure for an online learning platform

## 🚧 Work in Progress

* Payment gateway integration
* Sending purchase confirmation codes via email
* Additional minor improvements and optimizations

---

This project is built using classic ASP.NET and does not use ASP.NET Core.
